# Carrito de Compras de Fontana Cakes

Este es un proyecto de carrito de compras web desarrollado utilizando las tecnologías React, Vite, React Bootstrap, React Router y Firebase. El carrito de compras está diseñado para la tienda "Fontana Cakes", donde los usuarios pueden explorar una amplia selección de deliciosos productos y agregarlos al carrito para su compra.

## Tecnologias utilizadas

- React JS
- React Bootstrap
- React Router
- Firebase

## Características

- Vista de Catálogo: Los usuarios pueden navegar por la amplia gama de productos ofrecidos por Fontana Cakes y ver información detallada sobre cada artículo.
- Carrito de Compras: Los usuarios pueden agregar o eliminar productos del carrito de compras, así como ajustar las cantidades de los productos existentes en el carrito.
- Registro y Autenticación de Usuarios: Los usuarios pueden crear cuentas, iniciar sesión y recuperar sus contraseñas utilizando Firebase Authentication.
- Proceso de Compra: Los usuarios pueden revisar los productos en su carrito, proporcionar información de envío y procesar el pago utilizando métodos de pago seguros y confiables.

## Instalación

Para ejecutar la aplicación localmente, sigue estos pasos:

### Clona el repositorio de GitHub:
git clone https://github.com/tuusuario/fontana-cakes.git

### Instala las dependencias:
cd fontana-cakes
npm install

### Inicia el servidor de desarrollo:
npm run dev

La aplicación se abrirá automáticamente en tu navegador en `http://localhost:3000`.
